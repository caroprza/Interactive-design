Frontend y backend

El frontend se refiere a todo el diseño que tiene nuestra página, el cual va enfocado en el usuario, pues es lo que lo ayuda a interactuar con esta.

Mientras que el backend se encarga de la funcionalidad que hay detrás del sitio, como el proceso de datos.

End user y client

The end user se refiere a los usuarios del sitio, los que harán uso de este.

En cambio, el cliente es aquel que contacta a la empresa de software para la realización del proyecto, ya sea una página para su negocio o alguna otra situación.

CDN architecture

Content Delivery Network o CDN, se refiere a una red que le brinda contenido al sitio web, es decir, en vez de guardar elementos en nuestro propio servidor, estos se dispersan en otros servidores y simplemente se integran al sitio web. Los tipos de elementos que se pueden alojar son imágenes, video y audio, hojas de estilo CSS, archivos javascript, entre otros.

HTML and CSS history

HyperText Markup Language o conocido como HTML, tiene orígenes en 1980, donde Tim-Berners Lee propone por primera vez un sistema para compartir documentos, sin embargo, sería hasta 1991 cuando se presenta un documento oficial explicando su funcionamiento. En 1994 nace el World Wide Web Consortium o el W3C con el propósito de regular HTML y la web. Posteriormente se publica HTML 2.0 en 1995, el cual se convierte en el estándar oficial.

En 1998 se estrena HTML 4.0, el cual viene a la mano con las hojas de estilo CSS, lo que da la posibilidad de incluir scripts en las páginas y mejorar el diseño del programa. Finalmente, en 2004 se crea una alianza entre Apple, Oracle y Opera, conocida como Web Hypertext Application Technology Working Group o WHATWG, con el propósito de desarrollar HTML 5, aunque sería hasta 2014 cuando se presenta su lanzamiento final.
